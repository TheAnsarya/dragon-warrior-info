üöÄ Quick Start

1. Extract Everything

Run all extraction tools:

python tools/organize_chr_tiles.py python tools/extract_all_data.py python tools/extract_maps_npcs.py python tools/extract_text_dialogs.py

2. Edit JSON Files

Modify extracted data in extracted_assets/data/:

  • monsters.json - Change HP, attack, defense, etc.
  • spells.json - Modify MP costs
  • items_equipment.json - Edit weapon/armor stats

3. Reinsert Changes

Use reinsertion tool to create modified ROM:

from tools.reinsert_assets import DragonWarriorROMModifier
modifier = DragonWarriorROMModifier("roms/Dragon Warrior (U) (PRG1) [!].nes", "output/modified.nes")
modifier.create_backup()
modifier.modify_monster_stats("extracted_assets/data/monsters.json")
modifier.save_modified_rom()

4. Test in Emulator

Load output/dragon_warrior_modified.nes in your NES emulator and enjoy!
